# 🎵 Drop your bansuri sample files HERE

This folder is where the **real flute sound** comes from. The website
works without it (it falls back to synthesis), but adding files here
makes it sound like a real bansuri.

## How to name files

Name each file by its **musical note**, using scientific pitch
(middle C = `C4`, which is the default "Sa"):

```
C4.mp3   D4.mp3   E4.mp3   F4.mp3   G4.mp3   A4.mp3   B4.mp3
C5.mp3   D5.mp3   E5.mp3   F5.mp3   G5.mp3   A5.mp3   B5.mp3
```

- Formats accepted: **.mp3**, **.ogg**, or **.wav**
- One clean, **sustained** single note per file (1–3 seconds is enough)
- Record at a steady volume, ideally without vibrato (the loop will hold it)

## How many do I need?

| Files | Result |
|-------|--------|
| **1** (e.g. `C4.mp3`) | Works — every note is pitch-shifted from it. Far notes sound a bit off. |
| **7** (`C4`–`B4`) | Good — one octave of real tone. |
| **14+** (`C4`–`B5`) | Best — realistic across the whole range. |

The more notes you provide, the less pitch-shifting is needed and the
more authentic it sounds. Recording your own bansuri = the most
authentic result (and a great portfolio story).
